# Minimalist E-commerce API Endpoints 

`Base URL: https://beminimalist.onrender.com`
## Authentication
> * POST /signup
> * POST /login
## Products
> * GET /products
> * GET /products/skin
> * GET /products/hair
> * GET /products/bathnbody
> * GET /products/:id
## Shopping Cart
> * GET /cart
> * POST /cart/add
> * POST /cart/remove
## User Address
> * POST /addaddress
> * GET /getaddress
## Personal Data
> * POST /adddetails
> * GET /getdetails
